\doxysubsection{Detailed Description}
Implementation of the elastic surface free energy function $\rho_0 \Psi^\mathrm{surf} = \dfrac{1}{8} \lambda^\mathrm{S} ( C^\mathrm{S}_{kk} - 2)^2 + \dfrac{1}{4} \mu^\mathrm{S} ( C^\mathrm{S}_{ki}C^\mathrm{S}_{ik} - 2 C^\mathrm{S}_{kk} + 2) $, where $C^\mathrm{S}_{kl} = F^\mathrm{S}_{ik} F^\mathrm{S}_{il}$, and $F^\mathrm{S}_{kl} = u^\mathrm{S}_{k,l} + \delta_{kl}- n_k n_l$. In the latter formula, $u^\mathrm{S}_{k,l}$ is the surface gradient of the vector field $u^\mathrm{S}_k$, which is defined on a surface. Furthermore, $n_k$ is the outward normal vector on the surface.
