\doxysubsection{Detailed Description}
Implementation of the dissipation function $\rho_0 \Delta^\eta = \eta d_{ij}d_{ij}$, where $\eta>0$ is a material parameter and $d_{ij}$ is symmetric.

Order of arguments\+:~\newline
 \mbox{[}0\mbox{]} $d^\mathrm{i}_{11}$~\newline

 \mbox{[}4\mbox{]} $2 d^\mathrm{i}_{23}$~\newline
