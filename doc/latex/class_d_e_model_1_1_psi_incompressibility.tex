\doxysubsection{Detailed Description}
Implementation of the Lagrangian multiplier function for incompressibility $\rho_0 \Psi^\mathrm{inc} = p(1-J) $, where $p$ is the Lagrangian multiplier and $J$ the determniation of a deformation measure.
